Diagnóstico de Ciberseguridad 🔒

1. ¿Qué significa el acrónimo **HTTPS**?

Protocolo seguro de transferencia de hipertexto
Protocolo de acceso privado a la red
Herramienta de escaneo de malware

2. ¿Cuál es una **buena práctica** de seguridad en contraseñas?

Usar la misma contraseña en todos los servicios
Usar combinaciones de letras, números y símbolos
Escribirla en un papel junto al computador

3. ¿Qué es el **phishing**?

Un tipo de ataque para robar información personal
Un método de autenticación segura
Un software antivirus

4. ¿Qué herramienta se usa comúnmente para **cifrar información**?

VPN
Firewall
AES

5. ¿Qué significa **2FA** en ciberseguridad?

Autenticación en dos factores
Sistema avanzado de firewall
Un tipo de malware

6. ¿Cuál de las siguientes **NO** es una buena práctica de seguridad?

Actualizar el software regularmente
Abrir correos desconocidos con enlaces sospechosos
Usar autenticación de dos factores (2FA)

7. ¿Cómo puedes protegerte contra **malware**?

Evitar descargas de fuentes desconocidas
Desactivar el firewall
Compartir contraseñas con amigos
